Thoughts on Web Accessibility

I believe Web Accessibility is very important because it allows all kinds of users to be able to use the web page. By focusing on Web Accessibility you allow for a more reputable web page as well as an overall better user experience. I think it is important to make accessible web sites because it shows how we are as developers. By making an accessible web site we show that we care about our user's experiences with our web site. It also demonstrates our work ethic as well.
